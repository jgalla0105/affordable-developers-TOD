Place TOD figures for the intro section in this folder.

The page will automatically display the first matching file it finds between the two TOD intro paragraphs if you name it one of these:

- `tod-intro.png`
- `tod-intro.jpg`
- `tod-intro.jpeg`
- `tod-intro.webp`
- `tod-intro.svg`

Recommended location:

- `static/figures/tod-intro.png`

Because this is inside `static/`, the file will be served directly in the built site.
